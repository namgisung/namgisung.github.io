## **open()**

open() 함수는 파일을 열기 위해 사용하는 함수이다.

파일을 읽거나 쓰기 전에 반드시 파일을 열어야 한다.

---

### **(1) 기본 구조**

```python
파일객체 = open("파일이름", "모드")
````

* 파일이름 : 열 파일의 경로
* 모드 : 파일 사용 방식

---

### **(2) 읽기 모드**

```python
f = open("test.txt", "r")
```

파일을 읽기 위해 연다.

---

### **(3) 쓰기 모드**

```python
f = open("test.txt", "w")
```

파일에 내용을 작성하기 위해 연다.

---

### **(4) 추가 모드**

```python
f = open("test.txt", "a")
```

파일 끝에 내용을 추가하기 위해 연다.

---

### **(5) 읽기/쓰기 모드**

```python
f = open("test.txt", "r+")
```

파일을 읽고 쓸 수 있다.

---
### **(6) 기타 모드들**

 모드    | 설명                      |
| ----- | ----------------------- |
| `r`   | 읽기                      |
| `w`   | 쓰기 (기존 내용 삭제)           |
| `a`   | 추가                      |
| `r+`  | 읽기 + 쓰기                 |
| `w+`  | 읽기 + 쓰기 (기존 내용 삭제)      |
| `a+`  | 읽기 + 추가                 |
| `rb`  | 바이너리 읽기                 |
| `rb+` | 바이너리 읽기 + 쓰기            |
| `wb+` | 바이너리 쓰기 + 읽기 (기존 내용 삭제) |
| `ab`  | 바이너리 추가                 |
| `ab+` | 바이너리 추가 + 읽기            |


#### w+

```python
f = open("test.txt", "w+")
```

* 읽기 + 쓰기
* 기존 파일 내용 삭제
* 파일이 없으면 새로 생성

---

#### a+

```python
f = open("test.txt", "a+")
```

* 읽기 + 추가
* 파일 포인터가 파일 끝에 위치
* 파일이 없으면 새로 생성

---

#### rb

```python
f = open("image.jpg", "rb")
```

* 바이너리 읽기
* 파일 포인터가 파일 시작 위치

---

#### rb+

```python
f = open("image.jpg", "rb+")
```

* 바이너리 읽기 + 쓰기

---

#### wb+

```python
f = open("image.jpg", "wb+")
```

* 바이너리 쓰기 + 읽기
* 기존 파일 내용 삭제
* 파일이 없으면 새로 생성

---

#### ab

```python
f = open("image.jpg", "ab")
```

* 바이너리 추가
* 파일 끝에 데이터 추가
* 파일이 없으면 새로 생성

---

#### ab+

```python
f = open("image.jpg", "ab+")
```

* 바이너리 추가 + 읽기
* 파일 끝에 포인터 위치
* 파일이 없으면 새로 생성

---

#### 참고: b(Binary)

```python
rb
wb
ab
```

의 `b`는 Binary(바이너리)를 의미한다.


### **(7) 파일 객체 확인**

```python
f = open("test.txt", "r")

print(f)
```

결과:

```text
<_io.TextIOWrapper name='test.txt' mode='r' encoding='UTF-8'>
```

open()은 파일 객체를 반환한다.

---

### **(8) 사용 후 close()**

```python
f = open("test.txt", "r")

f.close()
```

파일 사용이 끝나면 close()로 닫아주는 것이 좋다.

---

### 정리

* open()은 파일을 여는 함수이다.
* 파일 객체를 반환한다.
* 파일을 읽거나 쓰기 전에 반드시 사용한다.
* 사용이 끝나면 close()를 호출한다.
