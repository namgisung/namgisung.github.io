## **if**

if는 "만약"이라는 뜻으로 조건문에 쓰인다.

만약 ~라면, ~를 실행한다

---

### **(1) 기본 구조**

```python
if 조건식:
    # 실행문
```

```python
c = 0
if c == 0:
    print(c)
```

이처럼 if문 뒤에 조건식을 쓰고 콜론(`:`)을 붙인다.

그리고 그 조건이 참이라면 들여쓰기된 실행문이 실행된다.

[참고] 등호를 한번만 쓰면 왼쪽 변수에 오른쪽 값을 지정하는 것이고 등호를 두개쓰면 왼쪽과 오른쪽이 같다는 의미이다.

만일 조건이 거짓이라면 들여쓰기된 실행문은 동작하지 않고 if문을 빠져나간다.

---

## 추가 설명

Python에서는 중괄호 `{}` 대신 **들여쓰기**로 실행 범위를 구분한다.

```python
if True:
    print("실행됨")
```

들여쓰기가 맞지 않으면 오류가 발생한다.
