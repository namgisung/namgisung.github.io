## **딕셔너리**

딕셔너리는 키(key)와 값(value)을 한 쌍으로 저장하는 자료구조이다.

리스트가 인덱스를 이용해 값을 구분한다면, 딕셔너리는 키를 이용해 값을 구분한다.

예를 들어 학생의 이름과 나이를 저장하려면 다음과 같이 표현할 수 있다.

```python
student = {"name": "kim", "age": 20}
```

위 코드에서 "name"과 "age"는 키(key)이고,
"kim"과 20은 값(value)이다.

딕셔너리는 키를 통해 값을 빠르게 찾을 수 있다는 특징이 있다.

또한 리스트와 달리 순서가 아닌 “키”를 기준으로 데이터를 저장한다.
