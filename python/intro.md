python특징

1. 쉬운 문법: 영어와 비슷한 직관적인 문법

2. 높은 가독성: 들여쓰기를 통한 코드 구조화

3.  다양한 활용성: 웹 개발, 데이터 분석, 인공지능, 게임 개발 등

기본 구조
```python
# Python 기본 구조 예제

# 1. 변수 선언
name = "Python"
age = 10

# 2. 입력
user_name = input("이름을 입력하세요: ")
user_age = int(input("나이를 입력하세요: "))

# 3. 출력
print("언어 이름:", name)
print("나이:", age)

print(f"{user_name}님의 나이는 {user_age}살입니다")
```

#구성 요소

1. 변수 선언 → 값을 저장하기 위한 공간

2. 입력 → input() : 사용자 입력 받기

3. 출력 → print() : 값을 화면에 출력

4. 문자열 포맷팅 → f-string 문법, 문자열 안에 변수 값을 쉽게 삽입, 가독성이 좋고 많이 사용됨
