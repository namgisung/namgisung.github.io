## **튜플**

튜플은 여러 개의 값을 하나의 묶음으로 저장하는 자료형이다.
리스트와 비슷하지만, 값을 변경할 수 없는 불변(immutable) 객체이다.

예를 들어 여러 값을 하나로 묶어 저장할 때 다음과 같이 사용할 수 있다.

```python
t = (1, 2, 3)
```

튜플은 각 요소가 순서대로 저장되며, 인덱스를 이용해 접근할 수 있다.

---

### **튜플의 생성**

튜플은 소괄호 `()`를 사용하여 생성한다.

```python
t = (1, 2, 3)
```

#### **(1) 다양한 자료형 저장**

튜플은 여러 자료형을 함께 저장할 수 있다.

```python
t = (1, "hello", 3.14)
```

#### **(2) 요소가 하나인 튜플**

요소가 하나일 경우 반드시 쉼표를 포함해야 한다.

```python
t = (1,)   # 튜플
```

```python
t = (1)    # int형
```

#### **(3) 괄호 생략**

튜플은 괄호 없이도 생성할 수 있다.

```python
t = 1, 2, 3
```

---

### **튜플의 길이와 인덱스**

튜플의 각 요소는 인덱스를 통해 접근할 수 있다.

인덱스는 0부터 시작하며, 범위는 0부터 길이-1까지이다.

```python
t = (10, 20, 30)

print(t[0])  # 10
print(t[1])  # 20
```

---

### **튜플의 특징**

튜플은 한 번 생성되면 값을 변경할 수 없다.

```python
t = (1, 2, 3)
t[0] = 10   # 오류 발생
```

#### **(1) 불변 객체**

튜플은 불변 객체이므로 값을 수정, 추가, 삭제할 수 없다.

#### **(2) 인덱싱과 슬라이싱 가능**

```python
t = (1, 2, 3, 4)

print(t[1:3])  # (2, 3)
```

---

### **튜플의 활용**

튜플은 다음과 같은 경우에 주로 사용된다.

* 값이 변경되면 안 되는 데이터
* 좌표와 같은 고정 데이터
* 함수에서 여러 값을 반환할 때

```python
point = (10, 20)
```

---

### **튜플의 패킹과 언패킹**

여러 값을 하나의 튜플로 묶는 것을 패킹이라고 한다.

```python
t = 1, 2, 3
```

튜플에 저장된 값을 각각의 변수로 나누는 것을 언패킹이라고 한다.

```python
a, b, c = t

print(a)  # 1
print(b)  # 2
print(c)  # 3
```

---

### **튜플과 리스트의 차이**

튜플과 리스트의 가장 큰 차이는 변경 가능 여부이다.

* 리스트는 값을 변경할 수 있다.
* 튜플은 값을 변경할 수 없다.
