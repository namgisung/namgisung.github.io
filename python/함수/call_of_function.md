

## **함수의 호출**

정의한 함수는 함수 이름을 사용하여 호출할 수 있다.
함수를 호출하면 함수 내부에 작성된 코드가 실행된다.

---

### **(1) 기본 구조**

함수의 호출은 다음과 같이 함수 이름 뒤에 괄호 `()`를 붙여서 사용한다.

```python
함수이름()
```

---

### **(2) 기본 예제**

```python
def hello():
    print("Hello")

hello()
```

위의 코드에서 `hello()`를 호출하면 함수 내부의 코드가 실행되어 "Hello"가 출력된다.

---

### **(3) 여러 번 호출**

함수는 필요할 때 여러 번 호출할 수 있다.

```python
def hello():
    print("Hello")

hello()
hello()
hello()
```

위의 코드는 "Hello"를 세 번 출력한다.
