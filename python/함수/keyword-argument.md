## **키워드 인자**

키워드 인자는 함수 호출 시 매개변수의 이름을 지정하여 값을 전달하는 방식이다.
매개변수의 순서에 상관없이 값을 전달할 수 있다.

---

### **(1) 기본 구조**

```python
함수이름(매개변수=값)
```

---

### **(2) 기본 예제**

```python
def func(a, b):
    print(a, b)

func(a=10, b=20)
```

위의 코드에서 매개변수 이름을 직접 지정하여 값을 전달한다.

---

### **(3) 순서 변경 가능**

키워드 인자는 순서에 상관없이 사용할 수 있다.

```python
func(b=20, a=10)
```

위와 같이 순서를 바꾸어도 정상적으로 실행된다.

---

### **주의사항**

함수 호출 시
**일반 인자 다음에 키워드 인자를 작성해야 한다**

```python
func(10, b=20)   # 가능
func(a=10, 20)   # 오류
```
