## **함수의 정의**

함수는 `def` 키워드를 사용하여 정의한다.
함수는 특정 기능을 수행하는 코드의 묶음이며, 필요할 때 호출하여 사용할 수 있다.

---

### **(1) 기본 구조**

```python
def 함수이름():
    # 실행문
```

---

### **(2) 기본 예제**

```python
def hello():
    print("Hello")
```

위의 함수는 `hello`라는 이름의 함수를 정의한 것으로, 호출하면 "Hello"를 출력한다.

---

## **함수 이름 작성 규칙**

함수 이름은 해당 함수가 수행하는 동작을 나타내도록 작성하는 것이 좋다.
일반적으로 동사 형태로 작성한다.

```python
def print_hello():
    print("Hello")

def calculate_sum(a, b):
    return a + b
```

함수 이름만 보고도 어떤 기능을 수행하는지 알 수 있도록 작성해야 한다.

---

## 작성 규칙

* 소문자와 밑줄(_)을 사용하는 snake_case 방식 권장
* 의미 있는 이름 사용
* 동작을 나타내는 이름 사용 (동사 형태)

---

## 잘못된 예

```python
def a():
    pass
```

위와 같이 의미 없는 이름은 코드의 가독성을 떨어뜨린다.
