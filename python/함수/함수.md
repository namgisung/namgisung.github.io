## **함수**

함수는 특정 작업을 수행하는 코드의 묶음이다.
같은 작업을 반복할 때 코드를 재사용할 수 있도록 해준다.

예를 들어 같은 내용을 여러 번 출력해야 할 경우, 아래와 같이 반복해서 작성할 수 있다.

```python
print("Hello")
print("Hello")
print("Hello")
```

하지만 이러한 반복 코드는 비효율적이므로, 함수를 사용하여 한 번에 처리할 수 있다.
