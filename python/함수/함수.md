## **함수**

함수는 특정 작업을 수행하는 코드의 묶음이다.
같은 작업을 반복할 때 코드를 재사용할 수 있도록 해준다.

예를 들어 같은 내용을 여러 번 출력해야 할 경우, 아래와 같이 작성할 수 있다.

```python
print("Hello")
print("Hello")
print("Hello")
```

이와 같이 반복되는 코드는 비효율적이며, 함수로 만들어 한 번에 처리할 수 있다.

---

### **(1) 함수의 사용 이유**

함수를 사용하면 반복되는 코드를 줄이고, 코드의 재사용성을 높일 수 있다.

또한 프로그램을 구조적으로 작성할 수 있어 가독성이 좋아진다.

---

### **(2) 함수의 특징**

함수는 다음과 같은 특징을 가진다.

* 하나의 기능을 수행한다
* 필요할 때 여러 번 호출할 수 있다
* 입력값(매개변수)과 출력값(return)을 가질 수 있다
