## **매개변수와 인자**

함수는 값을 입력받아 처리할 수 있으며, 이때 사용하는 것이 매개변수와 인자이다.

* 매개변수(parameter): 함수를 정의할 때 사용하는 변수
* 인자(argument): 함수를 호출할 때 전달하는 값

---

### **(1) 기본 구조**

```python
def 함수이름(매개변수):
    # 실행문

함수이름(인자)
```

---

### **(2) 기본 예제**

```python
def add(a, b):
    print(a + b)

add(2, 3)
```

위의 코드에서
`a`, `b`는 매개변수이고,
`2`, `3`은 인자이다.

함수를 호출하면 인자의 값이 매개변수에 전달되어 실행된다.

---

### **(3) 여러 개의 매개변수**

함수는 여러 개의 매개변수를 가질 수 있다.

```python
def func(a, b, c):
    print(a, b, c)

func(1, 2, 3)
```
