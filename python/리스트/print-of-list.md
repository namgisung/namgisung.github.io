## **리스트의 출력**

### **(1) for문 활용**
```python
score = [100, 95, 80, 70, 60]

for i in range(len(score)):
    print(score[i])
```

### **(2) 리스트 전체 출력**
```python
score = [100, 95, 80, 70, 60]
print(score)
```
출력:
```python
[100, 95, 80, 70, 60]
```
