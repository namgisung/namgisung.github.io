## **리스트 슬라이싱**

슬라이싱은 리스트의 일부분을 잘라내어 새로운 리스트를 만드는 방법이다.

---

### **(1) 기본 구조**

```python
리스트이름[시작:끝]
```

* 시작 인덱스부터
* 끝 인덱스 **직전까지** 가져온다

---

### **(2) 기본 예제**

```python
arr = [1, 2, 3, 4, 5]

print(arr[1:4])   # [2, 3, 4]
```

---

### **(3) 생략 가능**

```python
arr = [1, 2, 3, 4, 5]

print(arr[:3])   # [1, 2, 3]
print(arr[2:])   # [3, 4, 5]
print(arr[:])    # 전체 복사
```

---

### **(4) 음수 인덱스**

```python
arr = [1, 2, 3, 4, 5]

print(arr[-3:])   # [3, 4, 5]
```

---

### **(5) 간격(step) 포함 구조**

슬라이싱은 간격(step)을 포함하여 사용할 수 있다.

```python
리스트이름[시작:끝:간격]
```

---

### **(6) 간격(step) 예제**

```python
arr = [1, 2, 3, 4, 5, 6, 7, 8, 9]

print(arr[2:7:2])   # [3, 5, 7]
```

인덱스 2부터 6까지 중에서 2칸씩 건너뛰며 가져온다

---

### **(7) 다양한 step 활용**

```python
print(arr[::2])    # [1, 3, 5, 7, 9]
print(arr[1::2])   # [2, 4, 6, 8]
```

---

### **(8) 역순**

```python
print(arr[::-1])   # [9, 8, 7, 6, 5, 4, 3, 2, 1]
```

step을 -1로 하면 역순이 된다
