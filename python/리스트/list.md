## **리스트**
리스트란 여러 개의 값을 하나의 묶음으로 다루는 것이다.

예를 들어 학생 5명의 점수를 저장하기 위해서 5개의 변수를 선언해야 할 것이다.
```python
score1 = 50
score2 = 60
score3 = 70
score4 = 80
score5 = 90
```
변수 대신 리스트를 사용하면 다음과 같이 간단히 처리할 수 있다.
```python
score = [50, 60, 70, 80, 90]
```
리스트는 각 요소가 순서대로 저장되며, 인덱스를 통해 접근할 수 있다.
