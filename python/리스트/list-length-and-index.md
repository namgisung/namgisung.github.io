## **리스트의 길이와 인덱스**

리스트의 각 저장공간을 '요소'라고 하며
`리스트이름[인덱스]` 형식으로 접근한다.

---

### **(1) 인덱스**

인덱스는 각 요소를 구별하는 번호이다.

인덱스의 범위는 0부터 '리스트길이-1'까지이다.

```python
score = [50, 60, 70, 80, 90]
```

위 리스트의 인덱스는 0, 1, 2, 3, 4이다.

---

### **(2) len(리스트이름)**

```python
score = [50, 60, 70, 80, 90]
tmp = len(score)  # 5
```

`len()` 함수는 리스트의 길이를 반환한다.

Python 리스트는 길이를 자유롭게 변경할 수 있다.

---

### **(3) index(값)**

리스트에서 특정 값의 인덱스를 찾을 때는 `index()` 메서드를 사용한다.

```python
score = [50, 60, 70, 80, 90]

idx = score.index(70)
print(idx)   # 2
```

`index()`는 해당 값이 처음으로 나타나는 위치의 인덱스를 반환한다.

---

### **주의사항**

리스트에 해당 값이 없으면 오류가 발생한다.

```python
score.index(100)   # 오류
```

-----

### **(4) 음수 인덱스**
파이썬에서는 음수 인덱스를 사용하여 뒤에서부터 요소에 접근할 수 있다. 


**파이썬** 만 가능하다.

- -1: 마지막 요소
- -2: 뒤에서 두 번째 요소

```python
score = [50, 60, 70, 80, 90]

print(score[-1])   # 90
print(score[-2])   # 80
```

리스트의 끝에서부터 접근할 때 유용하다.
