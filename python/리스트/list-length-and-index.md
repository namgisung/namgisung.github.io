## **리스트의 길이와 인덱스**

리스트의 각 저장공간을 '요소'라고 하며 '리스트이름[인덱스]' 형식으로 접근한다.

-----

### **(1) 인덱스**

인덱스는 각 요소를 구별하는 번호이다.

인덱스의 범위는 0부터 '리스트길이-1'까지이다.
```python
score = [50, 60, 70, 80, 90]
```
위 리스트의 인덱스는 0, 1, 2, 3, 4이다.

-----

(2) len(리스트이름)
```python
score = [50, 60, 70, 80, 90]
tmp = len(score)  # 5
```
len() 함수는 리스트의 길이를 반환한다.

Python 리스트는 길이를 자유롭게 변경할 수 있다.
