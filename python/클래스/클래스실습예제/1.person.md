## **Person 클래스 실습**

Person 클래스를 정의하고 여러 객체를 생성하여 정보를 출력하는 실습이다.

---

### **실습 내용**

- Person 클래스 정의
- 인스턴스 필드 사용
- 생성자 __init__()
- 특수 메서드 __str__()
- 객체 리스트 저장

---

### **(1) Person 클래스 정의**

```python
class Person:

    def __init__(self, name, id, age, height, weight):
        self.name = name
        self.id = id
        self.age = age
        self.height = height
        self.weight = weight

    def __str__(self):
        return "%s %d세 %.1fcm %.1fkg" % (
            self.name,
            self.age,
            self.height,
            self.weight
        )
```

---

### **(2) 객체 생성 및 리스트 저장**

```python
persons = [

    Person("을지문덕", 1, 40, 180.0, 75.0),
    Person("계백", 2, 35, 175.0, 70.0),
    Person("김유신", 3, 50, 178.0, 72.0),
    Person("강감찬", 4, 45, 172.0, 68.0),
    Person("이순신", 5, 48, 181.0, 80.0)

]

for person in persons:
    print(person)
```
