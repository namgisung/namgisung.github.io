## **Circle 클래스 실습**

원의 반지름과 중심 좌표를 저장하는 Circle 클래스를 작성하는 실습이다.

---

### **실습 내용**

- 반지름 radius
- 중심 좌표 cx, cy
- 원의 넓이 계산
- 중심 좌표 출력

---

### **(1) Circle 클래스 정의**

```python
import math

class Circle:

    def __init__(self, radius, cx, cy):
        self.radius = radius
        self.cx = cx
        self.cy = cy

    def area(self):
        return math.pi * self.radius ** 2

    def center(self):
        return (self.cx, self.cy)
```

---

### **(2) 객체 생성 및 메서드 호출**

```python
c1 = Circle(5, 10, 20)

print("원의 넓이:", c1.area())
print("원의 중심:", c1.center())
```

---

### **실행 결과**

```text 
원의 넓이: 78.53981633974483
원의 중심: (10, 20)
```

---

### **(3) area() 메서드**

```python
def area(self):
    return math.pi * self.radius ** 2
```

원의 넓이를 계산하여 반환한다.

---

### **(4) center() 메서드**

```python

    return (self.cx, self.cy)
```

원의 중심 좌표를 튜플 형태로 반환한다.

---
