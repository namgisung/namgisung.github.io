## **Rectangle 클래스 실습**

사각형을 나타내는 Rectangle 클래스를 작성하는 실습이다.

두 사각형이 서로 겹치는지 확인하는 overlap() 메서드를 구현한다.

---

### **실습 내용**

- 사각형 좌표 저장
- 넓이 계산
- __str__()
- overlap() 구현
- 객체 비교

---

### **(1) Rectangle 클래스 정의**

```python
class Rectangle:

    def __init__(self, x, y, w, h):
        self.x = x
        self.y = y
        self.width = w
        self.height = h

    def __str__(self):
        return "(%d, %d, %d, %d)" % (
            self.x,
            self.y,
            self.width,
            self.height
        )

    def getArea(self):
        return self.width * self.height

    def overlap(self, r):

        if self.x + self.width < r.x:
            return False

        if r.x + r.width < self.x:
            return False

        if self.y + self.height < r.y:
            return False

        if r.y + r.height < self.y:
            return False

        return True
```

---

### **(2) 객체 생성**

```python
r1 = Rectangle(0, 0, 100, 100)
r2 = Rectangle(10, 10, 100, 100)
```

---

### **(3) 넓이 출력**

```python
print(r1.getArea())
```

결과:

```text
10000
```

---

### **(4) overlap() 사용**

```python
if r1.overlap(r2):
    print("r1과 r2는 서로 겹칩니다.")
else:
    print("r1과 r2는 겹치지 않습니다.")
```

결과:

```text
r1과 r2는 서로 겹칩니다.
```
