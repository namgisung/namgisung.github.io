## **Phonebook 클래스 실습**

딕셔너리를 사용하여 연락처를 저장하는 Phonebook 클래스를 작성하는 실습이다.

---

### **실습 내용**

- 딕셔너리 사용
- 연락처 저장
- add() 메서드 구현
- __str__() 구현
- 객체 내부 데이터 관리

---

### **(1) Phonebook 클래스 정의**

```python
class Phonebook:

    def __init__(self):
        self.contacts = {}

    def add(self, name, mphone=None, ophone=None, email=None):

        self.contacts[name] = {
            "mphone": mphone,
            "ophone": ophone,
            "email": email
        }

    def __str__(self):

        result = ""

        for name, info in self.contacts.items():

            result += "%s\n" % (name)

            if info["mphone"]:
                result += "Mobile Phone: %s\n" % (
                    info["mphone"]
                )

            if info["ophone"]:
                result += "Office Phone: %s\n" % (
                    info["ophone"]
                )

            if info["email"]:
                result += "Email Address: %s\n" % (
                    info["email"]
                )

            result += "\n"

        return result
```

---

### **(2) 객체 생성 및 연락처 추가**

```python
obj = Phonebook()

obj.add(
    "Kim",
    ophone="1234",
    email="kim@address.com"
)

obj.add(
    "Park",
    ophone="5678",
    email="park@address.com"
)

print(obj)
```

---

### **실행 결과**

```text

Office Phone: 1234
Email Address: kim@address.com

Park
Office Phone: 5678
Email Address: park@address.com
```
