## **Dice 클래스 실습**

주사위를 나타내는 Dice 클래스를 작성하는 실습이다.

roll() 메서드를 사용하여 주사위를 던지고 나온 숫자를 출력한다.

---

### **실습 내용**

- Dice 클래스 정의
- random.randint() 사용
- 메서드 구현
- 난수 생성

---

### **(1) Dice 클래스 정의**

```python
import random

class Dice:

    def roll(self):
        face = random.randint(1, 6)
        print(face)
```

---

### **(2) 객체 생성 및 메서드 호출**

```python 
d1 = Dice()

d1.roll()
d1.roll()
d1.roll()
```

---

### **실행 결과**

```text
3
6
1
```

실행할 때마다 결과는 달라질 수 있다.

---

### **(3) random.randint()**

```python
random.randint(1, 6)
```

1부터 6 사이의 정수를 랜덤으로 생성한다.

---

### **(4) roll() 메서드**

```python
def roll(self):
    face = random.randint(1, 6)
    print(face)
```

주사위를 던진 결과를 출력한다.
