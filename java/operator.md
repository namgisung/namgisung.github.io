**(1) 용어**

연산자: 연산을 수행하는 기호(+,-,*,/ 등)

피연산자: 연산자의 작업 대상(변수, 상수, 수식)
```java
x + 3
```
연산자: +, 피연산자: x, 3

**(2) 사칙 연산자 + - * /**

사칙 연산자는 아마도 프로그래밍에서 가장 많이 사용되는 연산자들 일 것이다. 피연산자가 정수인 경우, 나누는 수로 0을 사용할 수 없다.

**(3) 할당 연산자**
```java
a=b
```

**(4) 비교 연산자**
```java
a<b
a>b
a<=b
a>=b
```
**(5) 등가비교 연산자**

두 값이 같으면 true 아니면 false
```java
a==b 
```
두 값이 다르면 true 아니면 false
```java
a!=b
```
