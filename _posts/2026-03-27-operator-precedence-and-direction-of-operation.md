## 연산자 우선순위

### **괄>멤>단>산>시>관>비>논>삼>대>콤**

연산자|종류|방향
|---|-----|---
괄호 연산자|(),[],{}|좌->우
멤버 접근 연산자|[]|좌->우
단항 연산자(후위)|++,--(후위)|좌->우
단항 연산자(전위)|++,--,sizeof,!,~+,-,&,*|우->좌
산술 연산자|*,/,%,+,-|좌->우
시프트 연산자|<<,>>|좌->우
관계 연산자|<,>,<=,>=|좌->우
비트 연산자|&,^,\||좌->우
논리 연산자|&&,\|\||좌->우
삼항 연산자|?,:|좌->우
대입 연산자|=,+=,-=,*=,/=|우->좌
콤마 연산자|,|좌->우









--------

### 괄호 연산자
- (), [], {}

```text
5 * (3+7)
```

연산방향: 좌에서 우로

---------

### 멤버 접근 연산자
- []
```c
int a[5] = {1,2,3,4,5};
int b = a[3] + 4
```

연산방향: 좌에서 우로

---------

### 단항 연산자
-	++, – (전위 / 후위)
-	sizeof
-	! (논리 NOT)
-	~ (비트 NOT)
-	+, - (단항)
-	& (주소)
-	*	(역참조)
```c
int i = 3;

int a = i++;   // a = 3, i = 4
int b = ++i;   // i = 5, b = 5
int c = -i;    // c = -5
int d = sizeof(i); // d = 4 (보통 int 크기)
```

연산방향:

- 후위(++ , –) → 좌에서 우

- 전위 단항 → 우에서 좌

---------

### 산술 연산자
-	*, /, %
-	+, -
```c
int a = 3 + 4 * 5; // 23
```
연산방향: 좌에서 우

---------

### 시프트 연산자
-	<<, >>
```c
int a = 5 << 1; // 10
ㅑnt b = 8 >> 1; // 4
```
연산방향: 좌에서 우

---------

### 관계 연산자
-	<, <=, >, >=
```c
int a = 3 > 2; // 1 (true)
```
연산방향: 좌에서 우

---------

### 비트 연산자
-	&, ^, |
```c
int a = 5 & 3; // 1
int b = 5 | 3; // 7
```
연산방향: 좌에서 우

---------

### 논리 연산자
-	&&, ||
```c
int a = (3 > 2) && (2 > 1); // 1
```
연산방향: 좌에서 우

---------

### 삼항 연산자
-	? :

c언어, 자바에서
```c
int a = 5;
int b = (a > 3) ? 10 : 20; // 10
```
파이썬에서
```python
a= 5
b = 10 if a > 3 else 20
```

[C / Java]
- 삼항: 조건 ? 값1 : 값2

[Python]
- 삼항: 값1 if 조건 else 값2

연산방향: 우에서 좌

---------

### 대입 연산자
-	=, +=, -=, *=, /= 등
```c
int a;
a = 5;
a += 3; // 8
```
연산방향: 우에서 좌

---------

### 콤마 연산자
-	,
```c
int a = (1, 2, 3); // 3
```
연산방향: 좌에서 우

---------

※ 본 연산자 우선순위는 C/C++ 기준으로 정리됨

※ Python은 일부 문법(++, sizeof 등)이 다르지만
   연산 순서 개념은 동일함
