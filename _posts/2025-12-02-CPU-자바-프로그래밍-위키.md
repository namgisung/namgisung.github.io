### 코드 작성 전 주의 사항

- `public class` 부터 중괄호 전까지에 나오는 것은 **파일명**입니다.  
- Replit은 파일 이름이 `main.java`로 고정되어 있어 `public class Main`으로 바꿔야 합니다.  
- 만약 파일명이 다르다면, `public class 파일명`으로 작성하면 됩니다.

---

### 예제 코드

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, Java!");
    }
}
```
