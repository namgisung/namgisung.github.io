while문은 반복문으로 조건식이 참이면 실행문을 실행하는것을 반복한다

**(1) 기본 구조**

while문의 구조는 아래와 같다
```java
while (조건식) {
  //조건문이 참이면 수행될 문장들
}
```
**(2) 동작 순서**

위에 while문의 실행 순서는 다음과 같다

(1) 조건식이 참이면 {}안으로 들어가고, 거짓이면 while문을 벗어난다.

(2) {}의 문장을 수행하고 다시 조건식으로 들어간다.
```java
int i=1;
while (i<=10) {
  //실행문
  i++
}
```
위에 예시는 실행문을 10번 반복한다.
