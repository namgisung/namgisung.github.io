## C언어 for문

`for`문은 **반복 횟수를 알고 있을 때** 사용하기에 적합한 반복문이다.

구조가 조금 복잡해 보일 수 있지만, 반복의 시작과 끝이 한눈에 보여 **직관적**이다.

```c
for (int i = 1; i <= 5; i++) {
    printf("yes\n");
}
```

위 예시는 `"yes"`를 **5번 출력**한다.

---

## **(1) 기본 구조**

`for`문의 구조는 다음과 같다.

```c
for (초기화; 조건식; 증감식) {
    // 실행문
}
```

---

## **(2) 동작 순서**

`for`문은 조건식이 **참인 동안** `{}` 안의 문장들을 반복하다가,

조건식이 **거짓이 되면** 반복문을 벗어난다.

실행 순서는 다음과 같다.

1. 초기화 수행 (처음 한 번만 실행)

2. 조건식 검사

3. 조건식이 참이면 실행문 수행

4. 증감식 수행

5. 다시 조건식 검사 → 반복

6. 조건식이 거짓이면 `for`문 종료

---

## **(3) 초기화**

반복문에서 사용할 변수를 **초기화**하는 부분이며 **한 번만 실행**된다.

```c
int i = 0;
```

여러 변수를 동시에 초기화할 수도 있다.

```c
int i = 0, j = 10;
```

이때 **변수들의 타입은 같아야 한다.**

---

## **(4) 조건식**

조건식이 **참이면 반복을 계속**하고,

**거짓이면 반복을 중단**하고 `for`문을 빠져나간다.

`for`의 의미처럼 **“조건이 참인 동안” 반복**한다고 이해하면 된다.

```c
i < 10
```

---

## **(5) 증감식**

반복이 한 번 끝날 때마다 실행되는 부분이다.

```c
i++
i--
```

`i++`는 `i`를 1 증가시킨다는 뜻이다.

```c
// i++ == i = i + 1
```

`i--`는 `i`를 1 감소시킨다는 뜻이다.

```c
// i-- == i = i - 1
```

---

## **(6) 예제 코드**

```c
#include <stdio.h>

int main() {
    for (int i = 1; i <= 10; i++) {
        printf("%d\n", i);
    }
    return 0;
}
```

위 코드는 **1부터 10까지 출력**한다.

* `while` → 조건 중심
* `do-while` → 최소 1회 실행
* `for` → **반복 횟수 명확할 때**
