## do-while문

`do-while`문은 **반복문**으로, 조건식이 참이면 실행문을 반복한다.
`while`문과 달리 **조건을 나중에 검사**하기 때문에 **최소 한 번은 반드시 실행**된다.

---

### **(1) 기본 구조**

`do-while`문의 구조는 아래와 같다.

```c
do {
    // 조건식이 참이면 반복해서 수행될 문장들
} while (조건식);
```

`while (조건식);` 뒤에 **세미콜론(;)이 반드시 필요하다.**

---

### **(2) 동작 순서**

`do-while`문의 실행 순서는 다음과 같다.

1. 먼저 `do {}` 안의 문장을 **무조건 한 번 실행**한다.

2. 실행이 끝난 후 `while (조건식)`을 검사한다.

* 조건식이 **참이면** 다시 `do {}`로 돌아간다.
* 조건식이 **거짓이면** `do-while`문을 벗어난다.

---

### **(3) 예제 코드**

```c
#include <stdio.h>

int main() {
    int i = 1;

    do {
        // 실행문
        i++;
    } while (i <= 10);

    return 0;
}
```

---

### **(4) 코드 설명**

* `i`는 1부터 시작한다.
* `do` 블록 안의 실행문이 **먼저 실행**된다.
* 그 후 `i <= 10` 조건을 검사한다.
* 조건이 참인 동안 반복되며, 실행문은 **총 10번 실행**된다.

---

### **[while문과의 차이점]**

| 구분       | while문   | do-while문 |
| -------- | -------- | --------- |
| 조건 검사 시점 | 실행 **전** | 실행 **후**  |
| 최소 실행 횟수 | 0번 가능    | **1번 보장** |
| 세미콜론     | 없음       | **있음**    |


위키 구조상 다음으로 **for문** 넣으면 진짜 교과서 된다 😎
