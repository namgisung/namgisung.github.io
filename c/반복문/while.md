## while문

`while`문은 **반복문**으로, 조건식이 **참(true)** 이면 실행문을 계속 반복해서 실행한다.

---

### **(1) 기본 구조**

`while`문의 구조는 아래와 같다.

```c
while (조건식) {
    // 조건식이 참이면 수행될 문장들
}
```

---

### **(2) 동작 순서**

위의 `while`문의 실행 순서는 다음과 같다.

1️⃣ 조건식을 먼저 검사한다.

* 조건식이 **참이면** `{}` 안으로 들어간다.
* 조건식이 **거짓이면** `while`문을 벗어난다.

2️⃣ `{}` 안의 문장을 수행한 후, 다시 조건식으로 돌아간다.

이 과정을 조건식이 거짓이 될 때까지 반복한다.

---

### **(3) 예제 코드**

```c
#include <stdio.h>

int main() {
    int i = 1;

    while (i <= 10) {
        // 실행문
        i++;
    }

    return 0;
}
```

---

### **(4) 코드 설명**

* `i`는 1부터 시작한다.
* 조건식 `i <= 10`이 참인 동안 반복된다.
* 반복될 때마다 `i++`로 `i`의 값이 1씩 증가한다.
* 결과적으로 **실행문은 총 10번 반복**된다.
