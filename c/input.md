## **입력**

### **(1) 입력을 사용하기 위한 헤더 포함**

C언어에서 표준 입력을 사용하려면 `stdio.h` 헤더를 포함해야 한다.

```c
#include <stdio.h>
```

---

### **(2) 입력을 저장할 변수 선언**

C언어에서 입력은 **변수에 직접 저장되는 것이 아니라, 변수의 주소에 저장**된다.

따라서 입력을 받기 전에 **반드시 변수를 먼저 선언**해야 한다.

```c
int a;
```

---

### **(3) scanf를 이용한 입력**

C언어에서는 `scanf` 함수를 사용하여 입력을 받는다.

```c
scanf("%d", &a);
```

* `%d` → 입력받을 데이터의 **형식 지정자**
* `&a` → 입력된 값을 저장할 **변수 `a`의 메모리 주소**
* `&` → **주소 참조 연산자**

즉, 위 코드는

> “정수 형식의 데이터를 입력받아 변수 `a`가 저장된 메모리 공간에 저장한다”는 의미이다.

---

### **(4) 형식 지정자 (`%`)**

C언어에서는 입력 데이터의 타입이 고정되어 있지 않기 때문에
형식 지정자(`%`)를 사용하여 어떤 타입의 데이터를 입력받을지 명시해야 한다.

| 자료형      | 형식 지정자 |
| -------- | ------ |
| `int`    | `%d`   |
| `double` | `%lf`  |
| `char`   | `%c`   |
| 문자열      | `%s`   |

```c
double x;
scanf("%lf", &x);
```

---

### **(5) 문자열 입력**

#### 🔹 공백 없는 문자열

```c
char str[100];
scanf("%s", str);
```

* 공백 전까지만 입력받음
* 문자열에는 `&`를 붙이지 않는다

---

#### 🔹 공백 포함 문자열 (권장)

```c
char str[100];
fgets(str, sizeof(str), stdin);
```

* 공백 포함 가능
* 엔터(`\n`)까지 함께 입력됨

---

### **(6) 입력 후 출력**

```c
int a;
scanf("%d", &a);

printf("%d\n", a);
```

```c
char str[100];
fgets(str, sizeof(str), stdin);
printf("%s", str);
```

---

### **(7) 여러 개의 입력 받기**

```c
int a, b;
scanf("%d %d", &a, &b);

printf("%d %d\n", a, b);
```

#### 입력

```
5 5
```

#### 출력

```
5 5
```

## **주의사항 (중요)**

### ❗ scanf 뒤에 fgets를 바로 쓰면 문제가 생길 수 있다

```c
int a;
char str[100];

scanf("%d", &a);
fgets(str, sizeof(str), stdin); // 빈 줄 입력됨
```

#### 이유

* `scanf`는 **개행 문자(`\n`)를 입력 버퍼에 남김**
* `fgets`가 그 `\n`을 바로 읽어버림

---

### ✅ 해결 방법

```c
int a;
char str[100];

scanf("%d", &a);
getchar(); // 남아 있는 '\n' 제거
fgets(str, sizeof(str), stdin);
```

---

## **핵심 정리**

* C언어에서 입력은 **값을 받는 것이 아니라 메모리 주소에 값을 저장하는 과정**이다.
* `scanf`는 형식 지정자(`%`)와 주소 연산자(`&`)를 함께 사용한다.
* 문자열 입력에는 `fgets` 사용이 가장 안전하다.
* `scanf`와 `fgets`를 함께 사용할 경우 입력 버퍼에 주의해야 한다.
