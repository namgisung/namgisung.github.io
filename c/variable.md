## **변수**

### **(1) 기본 구조**

```c
int a;
```

이런 식으로 **정수형 변수 `a`를 선언**한다.

```c
int a = 10;
```

이런 식으로 **변수를 선언함과 동시에 값을 초기화**한다.

---

### **(2) 상수 변수 선언**

C언어에는 **`const`** 를 사용한다.
값이 한 번 정해지면 **변경할 수 없는 변수**다.

```c
const int a = 0;
```

---

## **(3) 강제 형 변환 (형변환, 캐스팅)**

변수 타입을 바꾸고 싶을 때 **강제 형 변환**을 사용한다.

형식:

```
(타입)값
```

```c
double input = 3.14;     // 실수형 변수
int intValue = (int)input; // 실수 → 정수 (소수점 버림)

printf("%d\n", intValue); // 출력: 3
```

C에서 **소수점은 자동으로 버려진다**

---

## **(4) 변수 기본 자료형 종류**

### 논리형

```c
#include <stdbool.h>

bool isTrue = true;
```

* `stdbool.h` 필요
* `true` / `false`
* 조건문에 사용

---

### 문자형

```c
char c = 'A';
```

* **한 문자만 저장**
* 반드시 `' '` (작은따옴표) 사용

---

### 문자열

C에는 `String` 타입이 없음
→ **문자 배열(`char[]`)로 표현**

```c
char str[] = "Hello";
```

* 문자열은 `" "` (큰따옴표)
* 마지막에 `\0`(널 문자) 자동 포함

---

### 정수형

```c
int num = 10;
```

---

### 실수형

```c
float f = 3.14f;
double d = 3.14;
```

* `float` → 뒤에 `f` 붙이는 게 정석
* `double` → 기본 실수형

---

## **char vs 문자열 차이**

```c
char c = 'A';        // 문자 1개
char s[] = "A";     // 문자열
```

| 구분 | char   | 문자열      |
| -- | ------ | -------- |
| 저장 | 1글자    | 여러 글자    |
| 표기 | `'A'`  | `"ABC"`  |
| 타입 | `char` | `char[]` |
