## 간접 참조 연산자(*)

간접 참조 연산자 `*`는

**포인터가 가리키는 메모리 주소에 저장된 값을 가져오거나 변경할 때 사용**한다.

즉,

> **“이 주소가 가리키는 곳에 있는 값”**
> 을 의미한다.

---

### 기본 개념

```c
*포인터변수
```

* 포인터 변수에 저장된 **주소를 따라가서**
* 그 주소에 있는 **실제 값에 접근**

---

### 기본 예제

```c
int a = 10;
int *p = &a;

printf("%d\n", *p);
```

출력:

```
10
```

설명:

* `a`  → 값 10
* `&a` → a의 주소
* `p`  → a의 주소 저장
* `*p` → 그 주소에 저장된 값 → 10

✔️ `*p`는 사실상 `a`와 같다.

---

### 값 변경하기

```c
int a = 10;
int *p = &a;

*p = 20;

printf("%d\n", a);
```

출력:

```
20
```

`*p = 20;`

= “p가 가리키는 주소에 20을 저장해라”
= `a = 20;`

---

### 선언에서의 * 와 사용에서의 * (중요❗)

```c
int *p;   // 포인터 선언
*p = 10;  // 간접 참조 (값 접근)
```

같은 `*`지만 **의미가 다르다**

| 위치  | 의미           |
| --- | ------------ |
| 선언부 | 포인터 변수 선언    |
| 사용부 | 간접 참조 (값 접근) |

---

### & 와 * 관계 정리

```c
int a = 10;
int *p = &a;
```

| 표현 | 의미 |
| -- | -- |
| a  | 값  |
| &a | 주소 |
| p  | 주소 |
| *p | 값  |

`&`는 **주소를 얻고**

`*`는 **주소를 따라간다**

---

### 함수에서 간접 참조 활용

```c
void change(int *x) {
    *x = 100;
}

int main() {
    int a = 10;
    change(&a);
}
```

* `&a` → a의 주소 전달
* `*x` → 원본 변수 a를 직접 수정

✔️ C언어에서 **함수로 값을 바꾸는 유일한 방법**

---

### 잘못된 사용 예

❌ 초기화 안 된 포인터 사용

```c
int *p;
*p = 10; // 위험 (어디를 가리키는지 모름)
```

❌ NULL 포인터 간접 참조

```c
int *p = NULL;
*p = 10; // 런타임 오류
```

✔️ 반드시 **유효한 주소를 가리킬 때만 `*` 사용**
