## **출력**

### **(1) `printf("")`**

`""` 안에 문자나 변수를 넣어 **출력**할 수 있다.

문자와 변수를 함께 출력할 때는 **형식 지정자(`%`)를 사용**한다.

문자열 안에 `\n`을 넣어 **줄바꿈**을 할 수 있다.

```c
printf("Hello, World!\n");
```

---

### **(2) `puts("")`**

`puts()`는 문자열을 출력한 뒤 **자동으로 줄바꿈**을 한다.

```c
puts("Hello, World!");
```

* 문자열 출력 전용
* 변수 출력이나 형식 지정자 사용 ❌

---

### **(3) `printf()` (형식화된 출력)**

```c
printf("출력 %d", a);
```

형식화된 출력 함수로,
앞에는 **출력할 문자열과 형식 지정자**를 쓰고
뒤에는 **출력할 변수들**을 순서대로 입력한다.

* `printf`는 **자동 줄바꿈을 하지 않는다**
* 줄바꿈이 필요하면 `\n`을 직접 작성해야 한다

---

### **(4) 형식 지정자**

| 지정자  | 의미                     |
| ---- | ---------------------- |
| `%d` | 10진수 정수                |
| `%f` | 실수 (`float`, `double`) |
| `%s` | 문자열                    |
| `%c` | 문자                     |
| `\n` | 줄바꿈                    |

```c
int a = 3;
double b = 3.14;
char str[] = "안녕";
char c = 'a';

printf("a = %d\n", a);
printf("b = %f\n", b);
printf("나는 %s\n", str);
printf("c = %c\n", c);
```

---

### **실수 출력 자리수 조절**

```c
printf("b = %.2f\n", b);
```

* 소수점 둘째 자리까지 출력
