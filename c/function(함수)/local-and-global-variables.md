## 지역변수와 전역변수

변수는 **선언된 위치**에 따라 사용할 수 있는 범위(유효 범위, scope)가 달라지며,

이에 따라 **지역변수**와 **전역변수**로 나뉜다.

---

### (1) 전역변수

전역변수(Global Variable)는 **함수 밖에서 선언된 변수**로,

프로그램 전체에서 사용할 수 있는 변수이다.

```c
#include <stdio.h>

int x = 10;   // 전역변수

void func() {
    printf("%d\n", x);
}

int main() {
    func();
    printf("%d\n", x);
    return 0;
}
```

#### 전역변수의 특징

* 모든 함수에서 접근 가능
* 프로그램 시작 시 생성되어 프로그램 종료 시까지 유지
* 여러 함수에서 값을 공유할 수 있음
* 과도하게 사용하면 코드의 가독성과 유지보수가 어려워질 수 있음

---

### (2) 지역변수

지역변수(Local Variable)는 **함수 또는 중괄호 `{ }` 내부에서 선언된 변수**로,

선언된 범위 안에서만 사용할 수 있다.

```c
#include <stdio.h>

void func() {
    int a = 5;   // 지역변수
    printf("%d\n", a);
}

int main() {
    func();
    // printf("%d\n", a);  // 오류 발생
    return 0;
}
```

#### 지역변수의 특징

* 선언된 함수나 블록 내에서만 사용 가능
* 함수가 종료되면 메모리에서 제거됨
* 다른 함수에서 같은 이름의 변수를 선언해도 서로 영향을 주지 않음

---

### 지역변수와 전역변수의 우선순위

지역변수와 전역변수의 이름이 같을 경우,

**지역변수가 전역변수보다 우선적으로 사용된다.**

```c
#include <stdio.h>

int num = 10;   // 전역변수

int main() {
    int num = 5;   // 지역변수
    printf("%d\n", num);  // 5 출력
    return 0;
}
```

---

### 지역변수와 전역변수 비교

| 구분    | 지역변수       | 전역변수    |
| ----- | ---------- | ------- |
| 선언 위치 | 함수 / 블록 내부 | 함수 밖    |
| 사용 범위 | 선언된 영역 내부  | 모든 함수   |
| 생명 주기 | 함수 실행 중    | 프로그램 전체 |
| 사용 목적 | 임시 계산      | 데이터 공유  |
