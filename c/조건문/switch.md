## **switch문**

`if`문은 조건식의 결과가 참과 거짓 두 가지밖에 없기 때문에
경우의 수가 많아질수록 `else if`를 계속 추가해야 한다.
이로 인해 조건식이 복잡해지고, 여러 조건을 비교해야 하므로 코드 가독성이 떨어질 수 있다.

이와 달리 `switch`문은 **하나의 조건식으로 여러 경우의 수를 처리**할 수 있으며,
표현이 간결하여 코드를 이해하기 쉽다.

따라서 **처리해야 할 경우의 수가 많을 때는 `if`문보다 `switch`문을 사용하는 것이 적절하다.**

---

### **(1) switch문 기본 구조**

`switch`문의 기본 구조는 다음과 같다.

```c
switch (조건식) {
    case 값1:
        // 조건식의 결과가 값1과 같을 경우 수행될 문장
        break;
    case 값2:
        // 조건식의 결과가 값2와 같을 경우 수행될 문장
        break;
    default:
        // 조건식의 결과와 일치하는 case가 없을 경우 수행될 문장
}
```

---

### **(2) switch문 동작 순서**

`switch`문은 **조건식을 먼저 계산**한 뒤,
그 결과와 일치하는 `case`문으로 이동한다.

해당 `case`문 아래의 문장들을 실행하며,
`break`문을 만나면 `switch`문 전체를 빠져나간다.

`break`문이 없을 경우,
아래에 있는 다른 `case`문까지 **연속해서 실행(fall-through)** 된다.

```c
int n = 2;

switch (n) {
    case 1:
        printf("1\n");
    case 2:
        printf("2\n");
    case 3:
        printf("3\n");
}
```

출력:

```
2
3
```

---

`switch`문은 항상 `if`문으로 변환할 수 있지만,
`if`문은 `switch`문의 제약 조건 때문에 `switch`문으로 바꿀 수 없는 경우가 많다.

---

### **(3) switch문 제약 조건**

`switch`문의 제약 조건은 다음과 같다.

#### (1) 조건식의 결과는 **정수형**이어야 한다

* `int`, `char`, `enum` 사용 가능
* **실수(`float`, `double`) 사용 불가**
* C언어에서는 **문자열 비교 불가**

```c
switch (num) { }      // 가능
switch (ch) { }       // 가능
switch (3.14) { }     // ❌ 불가능
```

---

#### (2) case문의 값은 **정수 상수**여야 한다

* 변수 사용 불가
* 중복된 값 사용 불가

```c
case 1:   // 가능
case 10:  // 가능
case a:   // ❌ 불가능
```
