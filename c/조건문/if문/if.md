## **if문**

`if`는 “만약”이라는 뜻으로 **조건문**에 사용된다.

> 만약 조건이 참이라면, 해당 코드를 실행한다.

C언어에서는 조건식의 결과가 **0이면 거짓**, **0이 아니면 참**으로 판단된다.

---

### **(1) 기본 구조**

```c
if (조건식) {
    // 실행문
}
```

```c
int c = 0;

if (c == 0) {
    printf("%d\n", c);
}
```

이처럼 `if`문 바로 뒤에 **조건식**을 작성하고,
그 뒤에 중괄호 `{}` 안에 **실행할 문장**을 작성한다.

조건식의 결과가 **참**이면 중괄호 안의 실행문이 수행된다.

---

### **조건이 거짓일 경우**

조건식이 거짓이면 중괄호 안의 실행문은 실행되지 않고,
`if`문을 그대로 빠져나간다.

```c
int c = 1;

if (c == 0) {
    printf("출력되지 않음\n");
}
```

---

### **[주의] 대입 연산자와 비교 연산자 차이**

```c
c = 0;   // 대입: 오른쪽 값을 왼쪽 변수에 저장
c == 0;  // 비교: 두 값이 같은지 비교
```

```c
if (c = 0)   // ❌ 항상 거짓
if (c == 0)  // ✅ 비교
```

* `=`  : 값 대입
* `==` : 값 비교
