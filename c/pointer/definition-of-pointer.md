## 포인터의 정의

**포인터(pointer)** 란

**다른 변수의 ‘값’이 아니라 ‘메모리 주소’를 저장하는 변수**이다.

일반 변수는 데이터를 직접 저장하지만,

포인터는 **데이터가 저장된 위치(주소)**를 저장한다는 점에서 차이가 있다.

---

### 변수와 포인터의 차이

```c
int a = 10;
```

* 변수 `a`는 값 `10`을 직접 저장한다.

```c
int *p = &a;
```

* 포인터 `p`는 변수 `a`의 **주소**를 저장한다.
* 즉, `p`는 `a`를 직접 가리킨다.

**값을 다루는 것이 아니라, 값이 있는 위치를 다룬다**는 것이 포인터의 핵심이다.

---

### 왜 포인터가 필요한가?

포인터를 사용하면 다음과 같은 작업이 가능해진다.

* 함수에서 여러 개의 값을 처리
* 배열과 문자열을 효율적으로 처리
* 메모리를 직접 제어
* 동적 메모리 사용 (`malloc`, `free`)

즉, 포인터는 **C언어가 메모리 구조를 직접 다루기 위해 반드시 필요한 개념**이다.
