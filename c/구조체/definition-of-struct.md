## 구조체의 정의

구조체(struct)는 **서로 다른 자료형의 변수들을 하나로 묶어 새로운 자료형을 만드는 것**이다.

C언어의 변수는 기본적으로

* `int`, `double`, `char` 처럼 **한 가지 타입의 값만 저장**할 수 있다.

하지만 실제 데이터는 여러 정보가 **한 덩어리**로 묶여야 하는 경우가 많다.

예를 들어, 학생 한 명의 정보를 저장하려면 다음과 같은 데이터가 필요할 수 있다.

* 학번 (int)
* 이름 (char 배열)
* 평균 점수 (double)

이걸 각각 따로 선언하면 관리가 매우 불편하다.

```c
int id;
char name[20];
double avg;
```

이 변수들이 **같은 학생 정보**라는 관계가 코드에 드러나지 않는다.

---

### 구조체의 필요성

구조체를 사용하면 관련 있는 데이터를 **하나의 단위로 묶어서 관리**할 수 있다.

```c
struct Student {
    int id;
    char name[20];
    double avg;
};
```

이제 `Student`라는 **새로운 자료형**이 만들어진 것이다.

---

### 구조체의 핵심 특징

서로 다른 자료형을 하나로 묶을 수 있다

사용자 정의 자료형이다

메모리상에 **연속된 공간**으로 저장된다

현실 세계의 개념을 코드로 표현하기 좋다

---

### 구조체 vs 배열

| 구분  | 배열           | 구조체          |
| --- | ------------ | ------------ |
| 자료형 | 동일한 타입만 가능   | 서로 다른 타입 가능  |
| 목적  | 여러 개의 같은 데이터 | 하나의 객체/정보 묶음 |
| 예시  | 점수 목록        | 학생, 좌표, 계좌   |

---

### 구조체는 이런 상황에 사용한다

* 학생 정보
* 좌표 (x, y)
* 날짜 (년, 월, 일)
* 계좌 정보
* 패킷, 설정값 묶음

**“이 데이터들은 하나다”** 라고 말하고 싶을 때 구조체를 쓴다.
