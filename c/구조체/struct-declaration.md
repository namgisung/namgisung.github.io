## 구조체 선언

구조체를 사용하려면 먼저 **구조체를 선언**해야 한다.

구조체 선언은 `struct` 키워드를 사용하며, **구조체의 설계도(자료형)를 만드는 단계**라고 보면 된다.

### (1) 기본 구조

```c
struct 구조체이름 {
    자료형 멤버이름;
    자료형 멤버이름;
};
```

* `struct` : 구조체를 선언할 때 사용하는 키워드
* 구조체이름 : 구조체를 구분하기 위한 이름
* 멤버 : 구조체 안에 포함된 변수들

구조체 선언 끝에는 **반드시 세미콜론(;)** 이 붙는다.

---

### (2) 구조체 선언 예제

학생 정보를 저장하는 구조체를 선언하면 다음과 같다.

```c
struct Student {
    int id;         // 학번
    char name[20];  // 이름
    double avg;     // 평균 점수
};
```

위 코드는 다음과 같은 의미를 가진다.

* `Student`라는 **새로운 사용자 정의 자료형**을 만든 것
* 아직 메모리는 할당되지 않음
* 구조체는 **변수를 만들기 위한 틀**일 뿐이다

즉, 이 단계에서는 **값을 저장할 공간이 생기지 않는다.**

---

### (3) 구조체 선언의 핵심 정리

✔️ 구조체는 **서로 다른 자료형을 묶을 수 있다**

✔️ 구조체 선언은 **메모리 할당이 아니다**

✔️ 실제 사용하려면 구조체 변수 선언이 필요하다
