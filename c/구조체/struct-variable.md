## 구조체 변수

구조체를 선언했다면, 이제 **구조체 변수를 선언**해서 실제로 데이터를 저장할 수 있다.

구조체 변수는 **선언된 구조체 자료형을 이용해 만드는 변수**이다.

---

### (1) 기본 구조

```c
struct 구조체이름 변수이름;
```

이때부터 **메모리가 실제로 할당**된다.

---

### (2) 구조체 변수 선언 예제

앞에서 선언한 `Student` 구조체를 이용하면 다음과 같다.

```c
struct Student s1;
```

* `struct Student` : 구조체 자료형
* `s1` : 구조체 변수 이름
* `s1`은 `id`, `name`, `avg`를 모두 포함하는 하나의 묶음 변수이다

---

### (3) 구조체 멤버 접근 (.) 연산자

구조체 변수의 각 멤버에 접근하려면 **점(.) 연산자**를 사용한다.

```c
s1.id = 1;
strcpy(s1.name, "Kim");
s1.avg = 4.2;
```

* `s1.id` : s1 구조체의 id 멤버
* `s1.name` : s1 구조체의 name 멤버
* `s1.avg` : s1 구조체의 avg 멤버

문자열은 `=`로 대입할 수 없기 때문에 `strcpy()`를 사용한다.

(`string.h` 필요)

```c
#include <string.h>
```

---

### (4) 선언과 동시에 초기화

구조체 변수는 선언과 동시에 초기화할 수도 있다.

```c
struct Student s2 = { 2, "Lee", 3.8 };
```

* 선언 순서대로 값이 멤버에 저장된다
* 멤버 순서를 반드시 지켜야 한다

---

### (5) 구조체 변수의 특징 정리

✔️ 구조체 변수는 **여러 개의 변수를 하나로 묶은 것**

✔️ 멤버 접근 시 **.(점) 연산자 사용**

✔️ 선언 시점에 **메모리 할당 발생**

✔️ 배열, 함수, 포인터와 함께 사용 가능
