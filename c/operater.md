## **연산자**

### **(1) 용어**

**연산자**: 연산을 수행하는 기호 (`+`, `-`, `*`, `/` 등)

**피연산자**: 연산자의 작업 대상 (변수, 상수, 수식)

```c
x + 3
```

* 연산자: `+`
* 피연산자: `x`, `3`

---

### **(2) 사칙 연산자 (`+ - * /`)**

사칙 연산자는 프로그래밍에서 가장 많이 사용되는 연산자이다.

```c
int a = 10;
int b = 3;

printf("%d\n", a + b); // 덧셈
printf("%d\n", a - b); // 뺄셈
printf("%d\n", a * b); // 곱셈
printf("%d\n", a / b); // 나눗셈
```

* 피연산자가 **정수형일 경우**, 나눗셈 결과도 정수
* **0으로 나누는 것은 불가능** (실행 오류)

```c
10 / 3 = 3
```

---

### **(3) 할당 연산자**

오른쪽 값을 왼쪽 변수에 저장한다.

```c
a = b;
```

복합 할당 연산자도 사용 가능하다.

```c
a += b;
a -= b;
a *= b;
a /= b;
```

---

### **(4) 비교 연산자**

두 값을 비교하여 **참 또는 거짓**을 판단한다.

```c
a < b
a > b
a <= b
a >= b
```

* 조건문(`if`, `while`)에서 주로 사용됨
* 결과값은 **0(false) 또는 1(true)**

---

### **(5) 등가 비교 연산자**

두 값이 같으면 참, 다르면 거짓

```c
a == b
```

두 값이 다르면 참, 같으면 거짓

```c
a != b
```

⚠️ 주의

```c
a = b   // 대입
a == b  // 비교
```

---

### 🔹 C언어에는 `boolean` 타입이 기본으로 없다

* 비교 결과는 `int` 값으로 반환됨
* `true` → 1
* `false` → 0

```c
if (a == b) {
    // 참
}
```
