## 배열의 초기화 (C언어)

C언어에서 배열은 **자동으로 초기화되지 않는다.**

배열을 선언만 하면 각 요소에는 **쓰레기 값**이 들어가 있을 수 있다.

따라서 배열을 사용하기 전에 **반드시 초기화**해 주는 것이 좋다.

```c
int score[5];   // 길이가 5인 int 배열 (값 보장 안 됨)
```

원하는 값을 저장하려면 각 요소마다 직접 값을 지정해야 한다.

```c
int score[5];
score[0] = 50;
score[1] = 60;
score[2] = 70;
score[3] = 80;
score[4] = 90;
```

---

## **(1) for문 활용**

배열의 길이가 큰 경우,

요소 하나하나에 값을 직접 넣는 것보다는 **for문을 사용하는 것이 효율적**이다.

```c
int score[5];

for (int i = 0; i < 5; i++) {
    score[i] = i * 10 + 50;
}
```

이 방법은 **일정한 규칙이 있는 값**을 저장할 때 적합하다.

---

## **(2) 배열의 선언과 초기화를 동시에 하기**

C언어에서는 배열을 **선언과 동시에 초기화**할 수 있다.

```c
int score[5] = {50, 60, 70, 80, 90};
```

또는 배열의 길이를 생략할 수도 있다.

```c
int score[] = {50, 60, 70, 80, 90};
```

이 경우, `{}` 안에 있는 요소의 개수에 따라

배열의 길이가 **자동으로 결정**된다.

---

## **(3) 부분 초기화**

배열의 일부만 초기화하면

초기화되지 않은 나머지 요소들은 **0으로 자동 초기화**된다.

```c
int score[5] = {50, 60};
```

결과:

```text
50 60 0 0 0
```

---

## **(4) 주의사항 (중요)**

❌ C언어에서는 **배열 선언 후 `{}`를 사용한 초기화가 불가능**하다.

```c
int score[5];
score = {50, 60, 70, 80, 90};   // ❌ 오류
```

✔️ 배열 초기화는 **선언과 동시에만 가능**하다.

| 구분               | 자바 | C언어 |
