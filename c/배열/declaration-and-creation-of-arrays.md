## 배열의 선언과 생성 (C언어)

C언어에서는 **배열의 선언과 생성이 동시에 이루어진다.**

자바처럼 선언과 생성을 나누거나 `new` 키워드를 사용하지 않는다.

---

### **(1) 배열 선언 = 생성**

C언어에서 배열을 선언하면

👉 그 즉시 **실제 저장 공간이 생성**된다.

```c
타입 배열이름[길이];
```

예시:

```c
int score[5];
```

위 코드는 **길이가 5인 int형 배열**을 생성하며,

`score[0]`부터 `score[4]`까지 총 5개의 저장 공간이 만들어진다.

---

### **(2) 자바와의 선언 방식 비교**

자바:

```java
int[] score;          // 배열 선언 (참조변수)
score = new int[5];   // 배열 생성 (실제 공간 생성)
```

C언어:

```c
int score[5];         // 선언과 동시에 생성
```

✔️ C언어에서는 배열 이름이 **참조변수**가 아니라

✔️ **연속된 메모리 공간의 시작 주소**를 의미한다.

---

### **(3) 선언과 동시에 초기화**

배열은 선언과 동시에 초기화할 수 있다.

```c
int score[5] = {10, 20, 30, 40, 50};
```

또는 일부만 초기화할 수도 있다.

```c
int score[5] = {10, 20};
```

이 경우 초기화되지 않은 나머지 요소들은 **0으로 자동 초기화**된다.

---

### **(4) C언어 배열의 특징 요약**

* 선언 = 생성
* `new` 키워드 없음
* 배열 크기 고정
* 실행 중 배열 크기 변경 불가
* 배열 이름은 시작 주소
